1. CONTEXTO Y VISION GENERAL DEL PROYECTO
================================================================================

1.1 Resumen ejecutivo
----------------------
'Al Rescate' es un videojuego cuyo nucleo jugable es un sistema de gachapon. Toda la
experiencia del jugador gira en torno a obtener mascotas e items mediante tiradas aleatorias
dentro de una sola escena hub. El proposito del juego es recaudar fondos para la Cruz Roja:
la tienda de gemas existe en la UI, pero no se integra con ninguna pasarela de pago real.
Cada intento de compra muestra un mensaje en la consola de Unity simulando el flujo, y deja
el sitio preparado para que en el futuro se conecte la pasarela oficial de la Cruz Roja.

El proyecto se construye desde cero. No existe codigo previo a reutilizar. Todos los assets
(sprites, audio, animaciones) se crearan o se solicitaran ad hoc; este documento describe sus
dimensiones, formatos y configuracion de importacion para que el flujo de produccion sea
reproducible.

1.2 Stack tecnico definitivo
-----------------------------
Aspecto                  | Decision
-------------------------|----------------------------------------------------------
Motor                    | Unity 2022.3.62f2 LTS
Pipeline grafico         | Universal Render Pipeline 2D (URP 2D)
Lenguaje                 | C# (.NET Framework 4.x compatibility)
Input                    | New Input System (com.unity.inputsystem) - sin legacy
UI                       | uGUI con Canvas + TextMeshPro
Tweening (opcional)      | DOTween Free (asset gratuito)
Audio                    | Audio Mixer nativo + AudioSource pool propio
Persistencia             | PlayerPrefs (rapido) + JSON en Application.persistentDataPath
Plataformas              | PC (Windows / macOS / Linux) y Android (movil)

1.3 Pilares de diseno
----------------------
- Coleccionabilidad: el jugador quiere completar la coleccion de mascotas. El gacha es la
  unica via para obtenerlas.
- Sentido de progreso: incluso las tiradas que no entregan una mascota legendaria entregan
  items utiles, gemas u otros recursos.
- Justicia percibida: el sistema de pity garantiza que ningun jugador se quede sin obtener
  un legendario despues de muchas tiradas.
- Espectaculo visual: la animacion de revelado se intensifica con la rareza, generando
  momentos memorables.
- Proposito social: la tienda comunica claramente que el jugador esta apoyando a la Cruz Roja.

1.4 Estilo visual y referencia
--------------------------------
La referencia principal para el flujo del gacha es Genshin Impact: el jugador entra a un
panel dedicado, ve la maquina/banner, presiona tirar x1 o x10, recibe una animacion de luz
que va revelando los objetos, y al final aparece una pantalla con todas las cartas obtenidas.
Cuando hay un legendario, la animacion previa cambia (haz dorado en lugar de azul), avisando
al jugador que algo grande esta por aparecer.

Para 'Al Rescate' la maquina gachapon es una capsula clasica de gachapones (esfera con base)
con luces, estilizada visualmente para encajar en un universo 2D 'cute'. La paleta de la
escena hub combina rojos calidos (referenciando a la Cruz Roja) con blancos y dorados de
acento, evitando saturar al jugador.

1.5 Flujo de juego de alto nivel
----------------------------------
1. El jugador abre el juego y aterriza en la escena MainMenu.
2. Pulsa 'Jugar' y entra a la escena GachaHub.
3. Si es la primera vez que abre el juego, recibe un paquete inicial de Gemas gratuitas.
4. En el hub ve su contador de Gemas, su mascota equipada flotando junto a la camara,
   y la maquina gachapon central.
5. Pulsa 'Tirar x1' o 'Tirar x10'. Si tiene Gemas suficientes, se descuentan y se ejecuta
   la tirada.
6. Aparece el panel de Resultado con la animacion correspondiente segun la rareza obtenida.
7. El jugador puede tirar de nuevo o cerrar el panel para volver al hub.
8. Desde el HUD puede abrir la Tienda (compra simulada de gemas) o la Coleccion (ver y
   equipar mascotas).
9. Todo se persiste automaticamente entre sesiones (gemas, inventario, pity).

2. CONFIGURACION INICIAL DEL PROYECTO UNITY
================================================================================

2.1 Crear el proyecto
----------------------
1. Abrir Unity Hub, ir a 'Installs' y verificar Unity 2022.3.62f2. Instalarlo si no existe.
2. Al instalarlo, anadir modulos: Windows Build Support (IL2CPP), Mac Build Support,
   Linux Build Support, Android Build Support (incluyendo OpenJDK, Android SDK & NDK).
3. En 'Projects' presionar 'New project'.
4. Seleccionar la plantilla 2D (URP). Importante: usar esta y NO '2D Built-in Renderer'.
5. Indicar nombre 'AlRescate' y carpeta destino. Pulsar 'Create project'.

2.2 Verificacion de URP 2D
----------------------------
1. Edit > Project Settings > Graphics: 'Scriptable Render Pipeline Settings' debe tener
   un asset URP.
2. Edit > Project Settings > Quality: cada nivel de calidad debe referenciar un Renderer
   URP 2D. Verificar que 'Render Pipeline Asset' este poblado.
3. Abrir el asset URP en el inspector. En 'Renderer List' debe figurar un asset de tipo
   2D Renderer. Si no, anadirlo: clic derecho en Project > Create > Rendering > URP >
   2D Renderer Data.
4. La camara principal debe tener Renderer Type = 'Custom' apuntando al 2D Renderer.

NOTA: Si se ve todo de color magenta, ejecutar Window > Rendering > Render Pipeline
Converter > Built-in to URP 2D para convertir automaticamente.

2.3 Paquetes obligatorios
--------------------------
Abrir Window > Package Manager, seleccionar 'Unity Registry' e instalar:

Paquete              | Version recomendada | Proposito
---------------------|---------------------|------------------------
Universal RP         | 14.0.x              | Pipeline 2D
Input System         | 1.7.x               | Input nuevo
TextMeshPro          | 3.0.x               | Texto vectorial
2D Sprite            | 1.0.x               | Sprite Editor
2D Animation         | 9.0.x               | Animacion 2D rigging
2D PSD Importer      | 8.0.x               | Importar PSD multicapa
Burst                | 1.8.x               | Optimizacion
Mathematics          | 1.2.x               | Tipos vectoriales
Cinemachine          | 2.9.x               | Camaras (shake)
Unity UI             | 1.0.x               | Sistema uGUI

2.3.1 Activar el New Input System
1. Tras instalar Input System, aceptar la activacion del nuevo backend; el editor reiniciara.
2. Edit > Project Settings > Player > Other Settings > Active Input Handling:
   valor = 'Input System Package (New)'.
3. Crear asset Input Actions: clic derecho > Create > Input Actions. Nombrar 'GameInputActions'.
4. Anadir Action Maps: 'UI' y 'Player'.
5. Marcar 'Generate C# Class' y aplicar; esto crea GameInputActions.cs automaticamente.

2.3.2 Plugins externos opcionales
- DOTween Free: tweening declarativo. Disponible gratis en el Asset Store.
  Tras importarlo: Tools > Demigiant > DOTween Utility Panel > Setup DOTween.

NOTA: DOTween es opcional. En este documento todos los efectos se entregan tambien con
coroutines puras.

4. DISENO FUNCIONAL DEL SISTEMA GACHA
================================================================================

4.1 Moneda interna: Gemas
--------------------------
- Es la unica forma de tirar del gacha. No hay tickets ni divisas alternativas.
- Se obtiene jugando: bonificaciones diarias (50 Gemas), logros (100-300 Gemas) y
  tiradas que devuelven gemas extra como item raro.
- Tambien se obtiene 'comprando' en la tienda (compras simuladas).
- Cada tirada x1 cuesta 160 Gemas.
- Cada tirada x10 cuesta 1.440 Gemas (10% de descuento respecto a 10 tiradas individuales).
- El primer paquete de bienvenida son 1.600 Gemas gratis (entregadas la primera vez).

4.2 Contenido obtenible
------------------------

4.2.1 Mascotas
- Son el contenido estrella. Cada mascota es unica en la coleccion.
- Si ya se posee y vuelve a salir, se convierte en Gemas equivalentes a su rareza.
- 5 mascotas iniciales, distribuidas entre rarezas.
- Cada mascota tiene: sprite estatico, animacion idle, animacion de obtencion, animacion
  equipada flotando, efecto de particulas y un powerup pasivo.
- Solo puede haber una mascota equipada a la vez. Se elige desde Coleccion.

4.2.2 Items de powerup temporal
- Items consumibles que dan ventajas durante una sesion. Acumulables.
- Aparecen en el inventario y pueden usarse desde la UI.
- Son mas comunes que las mascotas.
- Tienen rareza, lo que afecta el efecto que entregan.

4.3 Tasas globales de la pool
-------------------------------
Sub-pool          | Probabilidad global | Comentario
------------------|---------------------|------------------------------------------
Items de powerup  | 70%                 | Predominantes; flujo de recompensa positivo
Mascotas          | 30%                 | Reservadas para los momentos especiales

================================================================================
7. SISTEMA DE PITY (GARANTIA DE LEGENDARIO Y EPICO)
================================================================================

Tres mecanismos simultaneos:
- Soft pity para Legendario: a partir de la tirada 75 sin Legendario, la probabilidad
  crece linealmente +6 puntos porcentuales por cada tirada extra.
- Hard pity para Legendario: en la tirada 90 sin Legendario, la siguiente garantiza
  un Legendario.
- Pity de Epico: cada 10 tiradas, si no se obtuvo ningun Epico (o superior), la decima
  tirada garantiza al menos un Epico.

7.1 Algoritmo del Soft Pity (Legendario)
-----------------------------------------
probabilidadLegendarioBase = 0.006   // 0.6%
softPityInicio = 75
hardPity = 90

if (n + 1) >= hardPity:
    probLegendario = 1.0   // garantia absoluta
else if (n + 1) > softPityInicio:
    extra = ((n + 1) - softPityInicio) * 0.06   // +6% por tirada extra
    probLegendario = min(1.0, probabilidadLegendarioBase + extra)
else:
    probLegendario = probabilidadLegendarioBase

Si se obtiene un Legendario, el contador n se reinicia a 0.

7.2 Algoritmo del Pity de Epico
---------------------------------
pityEpicoUmbral = 10
if (m + 1) >= pityEpicoUmbral and rolledRarity == Comun or rolledRarity == Raro:
    rolledRarity = Epico

7.3 Persistencia del pity
---------------------------
Los contadores n (legendario) y m (epico) deben persistir entre sesiones.
La persistencia se realiza en PlayerPrefs y en el JSON principal.

7.4 Tirada multiple x10 y pity
--------------------------------
Cada tirada dentro del lote se evalua secuencialmente. El contador se actualiza despues
de cada tirada.
La tirada x10 garantiza al menos un objeto de rareza Raro o superior (si la simulacion
produce 10 Comunes, la decima se fuerza a Raro).

13.3 AudioManager.cs
---------------------
using UnityEngine;
using UnityEngine.Audio;
using System.Collections.Generic;

public class AudioManager : MonoBehaviour
{
    public static AudioManager Instance { get; private set; }

    [Header("Mixer")]
    public AudioMixerGroup musicGroup;
    public AudioMixerGroup sfxGroup;

    [Header("Library")]
    [SerializeField] private List<AudioEntry> sfxLibrary = new();
    [SerializeField] private AudioClip hubMusic;

    [Header("Pool")]
    [SerializeField] private int sfxPoolSize = 8;

    private Dictionary<string, AudioClip> _sfxDict;
    private List<AudioSource> _sfxPool;
    private AudioSource _musicSource;

    [System.Serializable]
    public class AudioEntry
    {
        public string key;
        public AudioClip clip;
        [Range(0f, 1f)] public float volume = 1f;
        [Range(0.5f, 2f)] public float pitch = 1f;
    }

    void Awake()
    {
        if (Instance != null && Instance != this) { Destroy(gameObject); return; }
        Instance = this;
        DontDestroyOnLoad(gameObject);

        _sfxDict = new Dictionary<string, AudioClip>();
        foreach (var e in sfxLibrary)
            if (!string.IsNullOrEmpty(e.key) && e.clip != null)
                _sfxDict[e.key] = e.clip;

        _musicSource = gameObject.AddComponent<AudioSource>();
        _musicSource.outputAudioMixerGroup = musicGroup;
        _musicSource.loop = true;
        _musicSource.playOnAwake = false;
        _musicSource.volume = 0.7f;

        _sfxPool = new List<AudioSource>();
        for (int i = 0; i < sfxPoolSize; i++)
        {
            var src = gameObject.AddComponent<AudioSource>();
            src.outputAudioMixerGroup = sfxGroup;
            src.playOnAwake = false;
            src.loop = false;
            _sfxPool.Add(src);
        }
    }

    public void PlayHubMusic()
    {
        if (hubMusic == null) return;
        _musicSource.clip = hubMusic;
        _musicSource.Play();
    }

    public void StopMusic() => _musicSource.Stop();

    public void Play(string key, float volumeScale = 1f, float pitch = 1f)
    {
        if (!_sfxDict.TryGetValue(key, out var clip))
        {
            Debug.LogWarning($"[AudioManager] Clave SFX no encontrada: {key}");
            return;
        }
        var src = GetFreeSource();
        src.clip = clip;
        src.volume = volumeScale;
        src.pitch = pitch;
        src.Play();
    }

    private AudioSource GetFreeSource()
    {
        foreach (var s in _sfxPool) if (!s.isPlaying) return s;
        return _sfxPool[0];
    }

    public void SetMasterVolume(float linear) => SetMixer("Volume_Master", linear);
    public void SetMusicVolume(float linear) => SetMixer("Volume_Music", linear);
    public void SetSfxVolume(float linear) => SetMixer("Volume_SFX", linear);

    private void SetMixer(string param, float linear)
    {
        var mixer = musicGroup.audioMixer;
        float dB = Mathf.Log10(Mathf.Clamp(linear, 0.0001f, 1f)) * 20f;
        mixer.SetFloat(param, dB);
    }
}

using System.IO;
using UnityEngine;

public static class SaveSystem
{
    private const string PREFS_KEY = "SaveData_v1";
    private static readonly string JSON_PATH =
        Path.Combine(Application.persistentDataPath, "savedata.json");

    public static SaveData Current { get; private set; } = new SaveData();

    public static void Save()
    {
        string json = JsonUtility.ToJson(Current, true);
        PlayerPrefs.SetString(PREFS_KEY, json);
        PlayerPrefs.Save();
        try { File.WriteAllText(JSON_PATH, json); }
        catch (System.Exception ex)
        { Debug.LogWarning($"[SaveSystem] No se pudo escribir JSON: {ex.Message}"); }
        Debug.Log("[SaveSystem] Guardado completo. Gemas=" + Current.gems);
    }

    public static void Load()
    {
        try
        {
            string json = null;
            if (File.Exists(JSON_PATH)) json = File.ReadAllText(JSON_PATH);
            else if (PlayerPrefs.HasKey(PREFS_KEY)) json = PlayerPrefs.GetString(PREFS_KEY);

            if (!string.IsNullOrEmpty(json))
            {
                Current = JsonUtility.FromJson<SaveData>(json);
                if (Current == null) Current = new SaveData();
            }
            else { Current = new SaveData(); }
        }
        catch (System.Exception ex)
        {
            Debug.LogError("[SaveSystem] Error al cargar: " + ex.Message);
            Current = new SaveData();
        }
        Debug.Log("[SaveSystem] Carga completada. Gemas=" + Current.gems);
    }

    public static void Reset() { Current = new SaveData(); Save(); }
}

14.5 PlayerInventory.cs
-------------------------
using System;
using System.Collections.Generic;
using UnityEngine;

public class PlayerInventory : MonoBehaviour
{
    public static PlayerInventory Instance { get; private set; }

    public event Action<int> OnGemsChanged;
    public event Action<PetData> OnPetObtained;
    public event Action<GachaItemData> OnItemObtained;
    public event Action<PetData> OnPetEquipped;

    [SerializeField] private GachaDatabase database;

    public int Gems
    {
        get => SaveSystem.Current.gems;
        private set
        {
            SaveSystem.Current.gems = Mathf.Max(0, value);
            OnGemsChanged?.Invoke(SaveSystem.Current.gems);
        }
    }

    public PetData EquippedPet { get; private set; }

    void Awake()
    {
        if (Instance != null && Instance != this) { Destroy(gameObject); return; }
        Instance = this;
        DontDestroyOnLoad(gameObject);
    }

    void Start()
    {
        if (!string.IsNullOrEmpty(SaveSystem.Current.equippedPetId))
        {
            EquippedPet = database.GetPetById(SaveSystem.Current.equippedPetId);
            if (EquippedPet != null) OnPetEquipped?.Invoke(EquippedPet);
        }
    }

    public bool TrySpendGems(int amount)
    {
        if (Gems < amount) { AudioManager.Instance.Play("sfx_ui_error"); return false; }
        Gems -= amount;
        SaveSystem.Save();
        return true;
    }

    public void AddGems(int amount)
    {
        if (amount <= 0) return;
        Gems += amount;
        SaveSystem.Save();
    }

    public bool HasPet(string petId) => SaveSystem.Current.ownedPetIds.Contains(petId);

    public void GrantPet(PetData pet, out int gemsEquivalent)
    {
        gemsEquivalent = 0;
        if (pet == null) return;
        if (HasPet(pet.id))
        {
            gemsEquivalent = pet.duplicateGemValue;
            AddGems(gemsEquivalent);
            return;
        }
        SaveSystem.Current.ownedPetIds.Add(pet.id);
        OnPetObtained?.Invoke(pet);
        SaveSystem.Save();
    }

    public void GrantItem(GachaItemData item, int qty = 1)
    {
        if (item == null) return;
        var stack = SaveSystem.Current.itemStacks.Find(s => s.itemId == item.id);
        if (stack == null)
        {
            stack = new SaveData.ItemStack { itemId = item.id, quantity = 0 };
            SaveSystem.Current.itemStacks.Add(stack);
        }
        stack.quantity += qty;
        OnItemObtained?.Invoke(item);
        SaveSystem.Save();
    }

    public void EquipPet(PetData pet)
    {
        if (pet == null || !HasPet(pet.id)) return;
        EquippedPet = pet;
        SaveSystem.Current.equippedPetId = pet.id;
        OnPetEquipped?.Invoke(pet);
        SaveSystem.Save();
        AudioManager.Instance.Play("sfx_pet_equip");
    }

    public int GetItemQty(string itemId)
    {
        var st = SaveSystem.Current.itemStacks.Find(s => s.itemId == itemId);
        return st?.quantity ?? 0;
    }
}

16. ALGORITMO DE PROBABILIDADES (PESO ACUMULADO + PITY)
================================================================================

16.1 Concepto: peso acumulado (weighted random)
-------------------------------------------------
Dadas N opciones con pesos w[i], W = sum(w[i]).
Para escoger:
1. Generar r = Random.Range(0f, W)
2. Recorrer opciones acumulando peso: cumulative += w[i].
   La primera opcion cuyo cumulative >= r es la elegida.

16.2 Pseudocodigo del algoritmo de tirada
------------------------------------------
funcion EjecutarTirada():
    n = pityLegendaryCounter
    m = pityEpicCounter

    // 1. Calcular probabilidad efectiva de Legendario
    if (n + 1) >= hardPity:
        rareza = Legendario
    else:
        pLeg = pBaseLegendario
        if (n + 1) > softPityInicio:
            pLeg = min(1, pBaseLegendario + ((n+1) - softPityInicio) * softPityIncrement)

        // 2. Construir tabla de pesos ajustada
        delta = pLeg - pBaseLegendario
        sumaResto = pCommon + pRare + pEpic
        factor = (sumaResto - delta) / sumaResto
        pesosAjustados = {
            Comun: pCommon * factor,
            Raro: pRare * factor,
            Epico: pEpic * factor,
            Legendario: pLeg
        }
        rareza = WeightedRandom(pesosAjustados)

    // 3. Aplicar pity de Epico
    if (m + 1) >= epicPityThreshold and rareza in {Comun, Raro}:
        rareza = Epico

    // 4. Determinar si es mascota o item
    esMascota = Random.value < petChance

    // 5. Escoger contenido concreto
    contenido = ListaPorRareza(rareza, esMascota).Random()

    // 6. Actualizar contadores
    if rareza == Legendario: pityLegendaryCounter = 0
    else: pityLegendaryCounter += 1

    if rareza == Epico or rareza == Legendario: pityEpicCounter = 0
    else: pityEpicCounter += 1

    return GachaResult { rareza, contenido, esMascota }

16.3 WeightedRandom.cs
-----------------------
using System.Collections.Generic;
using UnityEngine;

public static class WeightedRandom
{
    public static int PickIndex(IList<float> weights)
    {
        float total = 0f;
        for (int i = 0; i < weights.Count; i++) total += weights[i];
        float r = Random.value * total;
        float cumulative = 0f;
        for (int i = 0; i < weights.Count; i++)
        {
            cumulative += weights[i];
            if (r <= cumulative) return i;
        }
        return weights.Count - 1;
    }

    public static T Pick<T>(IList<T> options, IList<float> weights)
    {
        int idx = PickIndex(weights);
        return options[idx];
    }
}

using UnityEngine;

public class PitySystem
{
    private readonly GachaPool _pool;
    public PitySystem(GachaPool pool) { _pool = pool; }

    public int LegendaryCounter
    {
        get => SaveSystem.Current.pityLegendaryCounter;
        set => SaveSystem.Current.pityLegendaryCounter = value;
    }

    public int EpicCounter
    {
        get => SaveSystem.Current.pityEpicCounter;
        set => SaveSystem.Current.pityEpicCounter = value;
    }

    public float GetEffectiveLegendaryChance()
    {
        int n = LegendaryCounter;
        if ((n + 1) >= _pool.hardPity) return 1f;
        if ((n + 1) > _pool.softPityStart)
        {
            float extra = ((n + 1) - _pool.softPityStart) * _pool.softPityIncrement;
            return Mathf.Min(1f, _pool.pLegendary + extra);
        }
        return _pool.pLegendary;
    }

    public bool ShouldForceEpic() => (EpicCounter + 1) >= _pool.epicPityThreshold;

    public void OnRarityRolled(Rarity r)
    {
        if (r == Rarity.Legendary) { LegendaryCounter = 0; EpicCounter = 0; }
        else if (r == Rarity.Epic) { LegendaryCounter += 1; EpicCounter = 0; }
        else { LegendaryCounter += 1; EpicCounter += 1; }
    }
}

using System;
using System.Collections.Generic;
using UnityEngine;

public class GachaManager : MonoBehaviour
{
    public static GachaManager Instance { get; private set; }

    [SerializeField] private GachaPool pool;
    public GachaPool Pool => pool;

    private PitySystem _pity;
    public event Action<List<GachaResult>> OnPullCompleted;

    void Awake()
    {
        if (Instance != null && Instance != this) { Destroy(gameObject); return; }
        Instance = this;
        DontDestroyOnLoad(gameObject);
        _pity = new PitySystem(pool);
    }

    public bool TryPullX1(out GachaResult result)
    {
        result = default;
        if (!PlayerInventory.Instance.TrySpendGems(pool.singlePullCost)) return false;
        result = SinglePull();
        PlayerProfile.RegisterPull(1);
        SaveSystem.Save();
        OnPullCompleted?.Invoke(new List<GachaResult> { result });
        return true;
    }

    public bool TryPullX10(out List<GachaResult> results)
    {
        results = new List<GachaResult>();
        if (!PlayerInventory.Instance.TrySpendGems(pool.multiPullCost)) return false;
        bool anyRareOrAbove = false;
        for (int i = 0; i < 10; i++)
        {
            bool isLast = (i == 9);
            GachaResult r = SinglePull(forceAtLeastRare: isLast && !anyRareOrAbove);
            if (r.rarity != Rarity.Common) anyRareOrAbove = true;
            results.Add(r);
        }
        PlayerProfile.RegisterPull(10);
        SaveSystem.Save();
        OnPullCompleted?.Invoke(results);
        return true;
    }

    private GachaResult SinglePull(bool forceAtLeastRare = false)
    {
        Rarity rar;
        if ((_pity.LegendaryCounter + 1) >= pool.hardPity)
        {
            rar = Rarity.Legendary;
        }
        else
        {
            float pLeg = _pity.GetEffectiveLegendaryChance();
            float delta = pLeg - pool.pLegendary;
            float sumaResto = pool.pCommon + pool.pRare + pool.pEpic;
            float factor = (sumaResto - delta) / sumaResto;
            float[] weights = new float[]
            {
                pool.pCommon * factor, pool.pRare * factor,
                pool.pEpic * factor, pLeg
            };
            int idx = WeightedRandom.PickIndex(weights);
            rar = (Rarity)idx;

            if (_pity.ShouldForceEpic() && (rar == Rarity.Common || rar == Rarity.Rare))
                rar = Rarity.Epic;
            if (forceAtLeastRare && rar == Rarity.Common)
                rar = Rarity.Rare;
        }

        _pity.OnRarityRolled(rar);

        bool isPet = UnityEngine.Random.value < pool.petChance;
        var result = new GachaResult { rarity = rar, isPet = isPet };

        if (isPet)
        {
            var list = pool.GetPetsByRarity(rar);
            if (list == null || list.Count == 0) isPet = false;
            else
            {
                result.pet = list[UnityEngine.Random.Range(0, list.Count)];
                if (PlayerInventory.Instance.HasPet(result.pet.id))
                {
                    result.isDuplicate = true;
                    result.gemsAwarded = result.pet.duplicateGemValue;
                }
                PlayerInventory.Instance.GrantPet(result.pet, out _);
            }
        }
        if (!isPet)
        {
            result.isPet = false;
            var list = pool.GetItemsByRarity(rar);
            result.item = list[UnityEngine.Random.Range(0, list.Count)];
            PlayerInventory.Instance.GrantItem(result.item, 1);
        }
        return result;
    }
}

using UnityEngine;
using System.Collections.Generic;

public class GachaDatabase : MonoBehaviour
{
    [SerializeField] private GachaPool pool;

    private Dictionary<string, PetData> _petsById;
    private Dictionary<string, GachaItemData> _itemsById;

    void Awake()
    {
        _petsById = new Dictionary<string, PetData>();
        foreach (var list in new[] { pool.petsCommon, pool.petsRare, pool.petsEpic, pool.petsLegendary })
            foreach (var p in list)
                if (p != null && !string.IsNullOrEmpty(p.id)) _petsById[p.id] = p;

        _itemsById = new Dictionary<string, GachaItemData>();
        foreach (var list in new[] { pool.itemsCommon, pool.itemsRare, pool.itemsEpic, pool.itemsLegendary })
            foreach (var i in list)
                if (i != null && !string.IsNullOrEmpty(i.id)) _itemsById[i.id] = i;
    }

    public PetData GetPetById(string id) => _petsById.TryGetValue(id, out var p) ? p : null;
    public GachaItemData GetItemById(string id) => _itemsById.TryGetValue(id, out var i) ? i : null;
    public IEnumerable<PetData> AllPets() => _petsById.Values;
    public IEnumerable<GachaItemData> AllItems() => _itemsById.Values;
}

using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.UI;
using TMPro;

public class RevealAnimationController : MonoBehaviour
{
    [Header("Refs")]
    public GameObject canvasResult;
    public RectTransform resultRoot;
    public Image beamOfLight;
    public Transform cardContainer;
    public GameObject resultCardPrefab;
    public Button btnPullAgain, btnClose, btnSkip;

    [Header("Particles")]
    public ParticleSystem fxCommon, fxRare, fxEpic, fxLegendary;

    [Header("Beam colors")]
    public Color colorBlue = new Color(0.30f, 0.45f, 1f);
    public Color colorPurple = new Color(0.55f, 0.30f, 1f);
    public Color colorGold = new Color(1f, 0.85f, 0.10f);

    private bool _skipRequested;

    void Start()
    {
        btnSkip.onClick.AddListener(() => _skipRequested = true);
        btnClose.onClick.AddListener(Close);
    }

    public IEnumerator PlayX1(GachaResult r)
    {
        ClearCards();
        canvasResult.SetActive(true);
        SetBeam(r.rarity);
        yield return AnimateBeam(0.6f);
        AudioManager.Instance.Play("sfx_capsule_open");
        yield return RevealCard(r);
        ShowButtons();
    }

    public IEnumerator PlayX10(List<GachaResult> results)
    {
        ClearCards();
        canvasResult.SetActive(true);
        Rarity max = Rarity.Common;
        foreach (var rr in results) if (rr.rarity > max) max = rr.rarity;
        SetBeam(max);
        yield return AnimateBeam(1.2f);
        AudioManager.Instance.Play("sfx_capsule_open");
        btnSkip.gameObject.SetActive(true);
        _skipRequested = false;
        foreach (var rr in results)
        {
            yield return RevealCard(rr);
            if (_skipRequested) break;
        }
        if (_skipRequested) ShowAllCards(results);
        btnSkip.gameObject.SetActive(false);
        ShowButtons();
    }

    private IEnumerator RevealCard(GachaResult r)
    {
        var card = Instantiate(resultCardPrefab, cardContainer);
        var view = card.GetComponent<ResultCardView>();
        view.Setup(r);
        PlayParticleByRarity(r.rarity);
        AudioManager.Instance.Play(SfxByRarity(r.rarity));
        yield return new WaitForSeconds(DurationByRarity(r.rarity));
    }

    private void SetBeam(Rarity max)
    {
        if (beamOfLight == null) return;
        beamOfLight.color = max == Rarity.Legendary ? colorGold :
                            max == Rarity.Epic ? colorPurple : colorBlue;
    }

    private IEnumerator AnimateBeam(float duration)
    {
        beamOfLight.gameObject.SetActive(true);
        Vector3 from = new Vector3(1f, 0f, 1f);
        Vector3 to = new Vector3(1f, 1f, 1f);
        float t = 0f;
        while (t < duration)
        {
            t += Time.deltaTime;
            beamOfLight.rectTransform.localScale = Vector3.Lerp(from, to, t / duration);
            yield return null;
        }
        beamOfLight.rectTransform.localScale = to;
    }

    private void PlayParticleByRarity(Rarity r)
    {
        switch (r)
        {
            case Rarity.Common: fxCommon.Play(); break;
            case Rarity.Rare: fxRare.Play(); break;
            case Rarity.Epic: fxEpic.Play(); break;
            case Rarity.Legendary: fxLegendary.Play(); break;
        }
    }

    private string SfxByRarity(Rarity r) => r switch
    {
        Rarity.Common => "sfx_reveal_common",
        Rarity.Rare => "sfx_reveal_rare",
        Rarity.Epic => "sfx_reveal_epic",
        Rarity.Legendary => "sfx_reveal_legendary",
        _ => "sfx_reveal_common"
    };

    private float DurationByRarity(Rarity r) => r switch
    {
        Rarity.Common => 0.5f, Rarity.Rare => 1.0f,
        Rarity.Epic => 1.6f, Rarity.Legendary => 3.5f, _ => 0.5f
    };

    private void ClearCards()
    {
        for (int i = cardContainer.childCount - 1; i >= 0; i--)
            Destroy(cardContainer.GetChild(i).gameObject);
        beamOfLight.rectTransform.localScale = new Vector3(1f, 0f, 1f);
        beamOfLight.gameObject.SetActive(false);
    }

    private void ShowAllCards(List<GachaResult> results)
    {
        ClearCards();
        foreach (var r in results)
        {
            var card = Instantiate(resultCardPrefab, cardContainer);
            card.GetComponent<ResultCardView>().Setup(r);
        }
    }

    private void ShowButtons()
    {
        btnPullAgain.gameObject.SetActive(true);
        btnClose.gameObject.SetActive(true);
    }

    private void Close()
    {
        canvasResult.SetActive(false);
        AudioManager.Instance.Play("sfx_ui_close");
    }
}

using System.Collections.Generic;
using UnityEngine;
using UnityEngine.UI;
using TMPro;

public class CollectionUI : MonoBehaviour
{
    [SerializeField] private GachaDatabase database;
    [SerializeField] private Transform gridContent;
    [SerializeField] private GameObject cellPrefab;
    [SerializeField] private CollectionDetailView detailView;

    [Header("Filters")]
    public Toggle togglePets, toggleItems;
    public Toggle toggleCommon, toggleRare, toggleEpic, toggleLegendary;

    void OnEnable() { Rebuild(); }

    public void Rebuild()
    {
        for (int i = gridContent.childCount - 1; i >= 0; i--)
            Destroy(gridContent.GetChild(i).gameObject);

        var inv = PlayerInventory.Instance;
        bool showPets = togglePets.isOn || (!togglePets.isOn && !toggleItems.isOn);
        bool showItems = toggleItems.isOn || (!togglePets.isOn && !toggleItems.isOn);

        if (showPets)
            foreach (var p in database.AllPets())
                if (PassesRarityFilter(p.rarity)) AddCellPet(p, inv.HasPet(p.id));

        if (showItems)
            foreach (var i in database.AllItems())
                if (PassesRarityFilter(i.rarity)) AddCellItem(i, inv.GetItemQty(i.id));
    }

    private bool PassesRarityFilter(Rarity r)
    {
        bool any = toggleCommon.isOn || toggleRare.isOn || toggleEpic.isOn || toggleLegendary.isOn;
        if (!any) return true;
        return (r == Rarity.Common && toggleCommon.isOn) ||
               (r == Rarity.Rare && toggleRare.isOn) ||
               (r == Rarity.Epic && toggleEpic.isOn) ||
               (r == Rarity.Legendary && toggleLegendary.isOn);
    }

    private void AddCellPet(PetData p, bool owned)
    {
        var go = Instantiate(cellPrefab, gridContent);
        go.GetComponent<CollectionCell>().SetPet(p, owned, () => detailView.ShowPet(p, owned));
    }

    private void AddCellItem(GachaItemData i, int qty)
    {
        var go = Instantiate(cellPrefab, gridContent);
        go.GetComponent<CollectionCell>().SetItem(i, qty, () => detailView.ShowItem(i, qty));
    }
}

17.7 CurrencyDisplay.cs
------------------------
using UnityEngine;
using TMPro;

public class CurrencyDisplay : MonoBehaviour
{
    public TextMeshProUGUI gemText;
    private int _displayValue;

    void OnEnable()
    {
        if (PlayerInventory.Instance != null)
        {
            PlayerInventory.Instance.OnGemsChanged += OnGemsChanged;
            _displayValue = PlayerInventory.Instance.Gems;
            gemText.text = _displayValue.ToString("N0");
        }
    }

    void OnDisable()
    {
        if (PlayerInventory.Instance != null)
            PlayerInventory.Instance.OnGemsChanged -= OnGemsChanged;
    }

    private void OnGemsChanged(int newValue)
    {
        StopAllCoroutines();
        StartCoroutine(AnimateTo(newValue));
    }

    private System.Collections.IEnumerator AnimateTo(int target)
    {
        float t = 0f, duration = 0.4f;
        int from = _displayValue;
        while (t < duration)
        {
            t += Time.deltaTime;
            int v = Mathf.RoundToInt(Mathf.Lerp(from, target, t / duration));
            gemText.text = v.ToString("N0");
            yield return null;
        }
        _displayValue = target;
        gemText.text = target.ToString("N0");
    }
}

17.8 PetController.cs
----------------------
using UnityEngine;

public class PetController : MonoBehaviour
{
    [SerializeField] private Transform spawnPoint;
    [SerializeField] private float floatAmplitude = 0.15f;
    [SerializeField] private float floatFrequency = 1.0f;

    private GameObject _instance;
    private Animator _animator;

    void OnEnable()
    {
        if (PlayerInventory.Instance != null)
            PlayerInventory.Instance.OnPetEquipped += SpawnPet;
        if (PlayerInventory.Instance != null && PlayerInventory.Instance.EquippedPet != null)
            SpawnPet(PlayerInventory.Instance.EquippedPet);
    }

    void OnDisable()
    {
        if (PlayerInventory.Instance != null)
            PlayerInventory.Instance.OnPetEquipped -= SpawnPet;
    }

    void SpawnPet(PetData pet)
    {
        if (_instance != null) Destroy(_instance);
        if (pet == null || pet.petPrefab == null) return;
        _instance = Instantiate(pet.petPrefab, spawnPoint.position, Quaternion.identity, spawnPoint);
        _animator = _instance.GetComponentInChildren<Animator>();
        if (_animator != null && pet.animatorController != null)
            _animator.runtimeAnimatorController = pet.animatorController;
        if (_animator != null) _animator.SetBool("IsEquipped", true);
    }

    void Update()
    {
        if (_instance == null) return;
        float y = Mathf.Sin(Time.time * floatFrequency * Mathf.PI * 2f) * floatAmplitude;
        _instance.transform.localPosition = new Vector3(0f, y, 0f);
    }
}

17.9 UIAnimator.cs (helper sin DOTween)

using System.Collections;
using UnityEngine;

public static class UIAnimator
{
    public static IEnumerator FadeIn(CanvasGroup cg, float duration = 0.3f)
    {
        cg.alpha = 0f;
        cg.gameObject.SetActive(true);
        float t = 0f;
        while (t < duration)
        {
            t += Time.deltaTime;
            cg.alpha = Mathf.Clamp01(t / duration);
            yield return null;
        }
        cg.alpha = 1f;
    }

    public static IEnumerator FadeOut(CanvasGroup cg, float duration = 0.2f, bool deactivateAtEnd = true)
    {
        float t = 0f, from = cg.alpha;
        while (t < duration)
        {
            t += Time.deltaTime;
            cg.alpha = Mathf.Lerp(from, 0f, t / duration);
            yield return null;
        }
        cg.alpha = 0f;
        if (deactivateAtEnd) cg.gameObject.SetActive(false);
    }

    public static IEnumerator ScalePop(RectTransform rt, float duration = 0.35f)
    {
        Vector3 from = Vector3.one * 0.6f;
        Vector3 to = Vector3.one;
        rt.localScale = from;
        float t = 0f;
        while (t < duration)
        {
            t += Time.deltaTime;
            float p = t / duration;
            float c1 = 1.70158f, c3 = c1 + 1f;
            float e = 1f + c3 * Mathf.Pow(p - 1, 3) + c1 * Mathf.Pow(p - 1, 2);
            rt.localScale = Vector3.LerpUnclamped(from, to, e);
            yield return null;
        }
        rt.localScale = to;
    }
}

20.1 Script de prueba de probabilidades
-----------------------------------------
using System.Collections.Generic;
using UnityEngine;

public class GachaProbabilityTester : MonoBehaviour
{
    public GachaPool pool;
    public int iterations = 100000;

    [ContextMenu("Run probability test")]
    void RunTest()
    {
        var counts = new Dictionary<Rarity, int>
        {
            { Rarity.Common, 0 }, { Rarity.Rare, 0 },
            { Rarity.Epic, 0 }, { Rarity.Legendary, 0 }
        };

        SaveSystem.Current.pityLegendaryCounter = 0;
        SaveSystem.Current.pityEpicCounter = 0;

        for (int i = 0; i < iterations; i++)
        {
            float[] w = new float[] { pool.pCommon, pool.pRare, pool.pEpic, pool.pLegendary };
            int idx = WeightedRandom.PickIndex(w);
            counts[(Rarity)idx]++;
        }

        foreach (var kv in counts)
        {
            float pct = (float)kv.Value / iterations * 100f;
            Debug.Log($"{kv.Key}: {kv.Value} ({pct:F2}%)");
        }
    }
}

22.6 RESUMEN EJECUTIVO DEL PROYECTO
================================================================================

'Al Rescate' es un sistema gacha completo en Unity 2022.3 LTS con URP 2D, construido desde
cero como nucleo de un videojuego de recaudacion para la Cruz Roja. El jugador tira de un
gachapon con la moneda Gemas, obtiene mascotas (5 iniciales) e items (8 iniciales), construye
una coleccion, equipa una mascota que lo acompana visualmente, y se beneficia de un sistema
de pity (soft 75 / hard 90 para Legendarios; 10 para Epicos) que garantiza un coleccionismo
justo. La tienda de gemas existe pero solo simula la transaccion; queda preparada para
conectar la pasarela real de la Cruz Roja.

La arquitectura es modular: GachaManager, PitySystem, PlayerInventory, AudioManager,
SaveSystem y los controladores de UI son piezas independientes que se comunican por eventos.
La hoja de ruta de 20 etapas guia la implementacion paso a paso, con criterios de verificacion
claros, y la persistencia ya esta resuelta tanto con PlayerPrefs como con JSON. Al final del
proceso se obtiene una build instalable en PC y Android, completamente funcional, con todos
los hooks listos para anadir gameplay adicional.
